# @ux4g/styles Package Summary

## Overview

This package provides production-ready CSS foundation styles for the UX4G Design System, built on top of `@ux4g/tokens`. It includes reset styles, typography, forms, status indicators, navigation patterns, and utility classes - all optimized for Indian government digital services.

## Layer Organization

### 1. Base Layer
**Purpose**: Foundation and browser normalization

**Files**:
- `reset.css` - CSS reset with accessibility enhancements
- `typography.css` - Typography system (headings, text, links)
- `surfaces.css` - Background surfaces and elevation
- `focus.css` - Focus indicators for keyboard navigation

**Import**:
```css
@import '@ux4g/styles/base';
```

### 2. Utilities Layer
**Purpose**: Reusable utility classes for composition

**Files**:
- `spacing.css` - Margin, padding, gap utilities
- `layout.css` - Flexbox, grid, positioning utilities

**Import**:
```css
@import '@ux4g/styles/utilities';
```

### 3. Components Layer
**Purpose**: Baseline styles for UI patterns

**Files**:
- `forms.css` - Form controls, labels, validation
- `status.css` - Alerts, badges, status indicators
- `navigation.css` - Headers, navigation, footers

**Import**:
```css
@import '@ux4g/styles/components';
```

## Token Dependencies

All styles consume design tokens from `@ux4g/tokens`:

```css
/* Typography */
--ux4g-font-family-*
--ux4g-font-size-*
--ux4g-font-weight-*
--ux4g-line-height-*

/* Colors */
--ux4g-color-text-*
--ux4g-color-surface-*
--ux4g-color-border-*
--ux4g-color-{status}

/* Spacing */
--ux4g-spacing-{0-20}

/* Effects */
--ux4g-shadow-*
--ux4g-radius-*
--ux4g-z-*
```

## Import Options

### Option 1: Complete System
```css
@import '@ux4g/styles';
```
Includes tokens + base + utilities + components

### Option 2: Specific Layers
```css
@import '@ux4g/styles/base';
@import '@ux4g/styles/utilities';
@import '@ux4g/styles/components';
```

### Option 3: Individual Files
```css
@import '@ux4g/styles/reset';
@import '@ux4g/styles/typography';
@import '@ux4g/styles/forms';
```

## Usage Examples

### React
```tsx
import '@ux4g/styles';

function App() {
  return (
    <div className="ux4g-container ux4g-py-12">
      <h1 className="ux4g-mb-6">Welcome</h1>
      <div className="ux4g-card">

      </div>
    </div>

```


## Naming Convention

All classes use the `.ux4g-` prefix:

```css
/* Components */
.ux4g-card
.ux4g-button
.ux4g-alert

/* Utilities */
.ux4g-m-4        /* margin */
.ux4g-p-6        /* padding */
.ux4g-flex       /* display: flex */
.ux4g-grid-cols-3

/* Variants */
.ux4g-alert-success
.ux4g-input-error

/* States */
.ux4g-nav-link-active
```

## Build Process

```bash
# Install dependencies
pnpm install

# Build package
pnpm run build

# Watch mode for development
pnpm run watch
```

**Build Steps**:
1. PostCSS Import - Resolve `@import` statements
2. PostCSS Nesting - Transform nested CSS
3. Autoprefixer - Add vendor prefixes
4. Output to `dist/`

## Integration Notes

### With @ux4g/tokens

`@ux4g/styles` depends on `@ux4g/tokens` and automatically imports token CSS:

```css
/* src/index.css */
@import '@ux4g/tokens/css';  /* Tokens loaded first */
@import './base/index.css';
@import './utilities/index.css';
@import './components/index.css';
```

### Theme Customization

Override tokens to customize the entire system:

```css
:root {
  --ux4g-color-primary: #1E3A8A;
  --ux4g-spacing-4: 1.25rem;
}

@import '@ux4g/styles';
```

### Indian Government Tricolor

Header includes national tricolor as accent:

```html
<header class="ux4g-header ux4g-header-government">
  <!-- Saffron, white, green gradient bottom border -->
</header>
```
