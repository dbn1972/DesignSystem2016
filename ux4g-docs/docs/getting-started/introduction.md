# UX4G Design System

A government-grade design system platform for building consistent, accessible digital public services across Indian government ministries, departments, and agencies.

## What is UX4G?

UX4G (User Experience for Government) is a comprehensive design system that provides:

- **77 documented components** with React, Angular, and Web Components implementations
- **Design tokens** for consistent styling across all frameworks
- **Service blueprints** for common government workflows (certificates, payments, identity)
- **India-specific components** — Aadhaar Input, PAN Card Input, UPI Payment Gateway, 22-language selector
- **Accessibility-first** design meeting WCAG 2.1 Level AA standards

## Who is it for?

| Audience | What they get |
|---|---|
| **React developers** | `@ux4g/react-core` component library with TypeScript |
| **Angular developers** | `@ux4g/angular-core` component library |
| **Web developers** | `@ux4g/web-components` for any framework |
| **Designers** | Design tokens, spacing scales, color system |
| **Accessibility reviewers** | WCAG specs, keyboard support, screen reader behavior |
| **Department leads** | Governance, conformance, onboarding guides |

## Key Principles

1. **Consistency** — Same components, same behavior across all government services
2. **Accessibility** — Every component meets WCAG 2.1 AA with keyboard and screen reader support
3. **Multi-framework** — React, Angular, and Web Components from a single source of truth
4. **India-first** — Built for Indian government context with Aadhaar, PAN, UPI, and 22 languages
5. **Research-backed** — Every component cites usability research and government digital service findings
