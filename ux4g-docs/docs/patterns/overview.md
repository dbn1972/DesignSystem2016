# Patterns

Reusable interaction patterns for common government service workflows.

## Identity Patterns

- **Sign In** — Email/mobile + password authentication
- **Sign Up** — New citizen registration with Aadhaar verification
- **OTP Verification** — Mobile and Aadhaar OTP flows
- **Forgot Password** — Account recovery with identity verification
- **Session Timeout** — Warning and auto-logout handling

## Service Patterns

- **Certificate Application** — Multi-step form with document upload
- **Status Tracking** — Application progress with timeline
- **Payment Flow** — UPI, Net Banking, Card payment processing
- **Document Upload** — File upload with validation and preview
- **Feedback Collection** — Post-service satisfaction rating

## Consent Patterns

- **Terms Acceptance** — Checkbox-based consent capture
- **Data Sharing Consent** — Granular permission management
- **Privacy Notice** — GDPR/IT Act compliant notices

## Dashboard Patterns

- **Citizen Dashboard** — Application list, notifications, profile
- **Officer Dashboard** — Case queue, review, approval workflow
