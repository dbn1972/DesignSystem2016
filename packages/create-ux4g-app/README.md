# create-ux4g-app

Scaffold a new project using the UX4G Government of India Design System.

## Usage

```bash
npx create-ux4g-app my-gov-app
npx create-ux4g-app my-gov-app --template react
npx create-ux4g-app my-gov-app --template angular
```

## Templates

- **react** (default) — Vite + React 18 + TypeScript + UX4G
- **angular** — Angular 18 + TypeScript + UX4G

## What you get

- Pre-configured project with UX4G dependencies
- TypeScript setup
- Build tooling (Vite for React, Angular CLI for Angular)
- Sample app with UX4G component examples
- Indian tricolor header styling

## After scaffolding

```bash
cd my-gov-app
npm install
npm run dev    # React
npm start      # Angular
```
